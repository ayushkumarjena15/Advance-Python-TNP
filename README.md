## 📖 About The Repository

Welcome to the **Advance Python TNP** repository! This collection is a meticulously organized hub of my advanced Python training, problem-solving, and practical session codes. It spans from core programming concepts to full-fledged mini-projects designed to mimic real-world software scenarios.

Whether you're looking for solutions to common logic problems, need a reference for Python basics, or want to explore Object-Oriented Programming (OOP) through guided mini-projects like a complete Banking System, you'll find it all right here!

---

## 🗂️ Folder Structure & Details

Here is the exact breakdown of each and every folder in this workspace:

### 🌟 Mini Projects
* **`Mini Project`**
  * Contains assorted small Python module projects (`mini_project_1.py` to `mini_project_4.py`) that implement basic real-world logic.
* **`Mini Project(Banking System)`**
  * A classic object-oriented programming project (`banking_system.py`) that handles simulated banking operations (create account, deposit, withdraw, check balance).
* **`Mini Project(Student Performance Analyser)`**
  * A data-driven script (`project.py`) to systematically evaluate student grades, performance metrics, and calculate total rankings.

### 📚 Practice Modules
* **`Practice-1`** to **`Practice-5`**
  * Dedicated directories for daily practice sessions. These folders contain codes for logical reasoning, loops, pattern creations, string and list manipulation (e.g., `grade_a_count.py`, `list_palindrome.py`, `movie.py`, `sort_alpha.py`).

### 📝 Assessments & Sessions
* **`Python Assessment`**
  * Includes 15 dedicated scripts (`no_1.py` to `no_15.py`) solving specific logical, algorithmic, and mathematical challenges formatted as a complete module test.
* **`Evening session Work`**
  * Practice scripts explicitly written during dedicated evening coding sessions.
* **`Function`**
  * Code focusing intensely on Python's functional programming, arguments, lambda functions, and scope.

### 📄 Root Files
* `Global.py` - Understanding local and global variable scopes in Python.
* `list.py` & `list_op.py` - Code implementing various list operations, comprehensions, slicing, and methods.
* `palindrome.py` - Core logic for identifying palindrome structures in strings or integers.
* `minor_project_2.py` - Additional project codebase residing at the root level.

## 🚀 How to Run

To run any script locally on your machine, just clone standard git repository and execute with Python 3.x:

```bash
# 1. Clone the repository
git clone https://github.com/ayushkumarjena15/Advance-Python-TNP.git

# 2. Navigate to the desired folder, e.g., the Banking System
cd "Advance Python TNP/Mini Project(Banking System)"

# 3. Run the script!
python banking_system.py
```
