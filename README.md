## 🚀 How to Run the React Dashboard

Want to test drive the ultra-smooth dashboard interface on your local machine?

```bash
# 1. Clone this repository
git clone https://github.com/ayushkumarjena15/Advance-Python-TNP.git

# 2. Navigate into the core folder
cd Advance-Python-TNP

# 3. Install NPM Core Dependencies
npm install

# 4. Fire up the Vite Development Server!
npm run dev
```
